# nutshell-act-on-sync
# nutshell-act-on-sync is created to make synchronization between nutshell and act-on account's entities People, Company, Leads and Contacts.
# Define your database connection in config/database.php file.
# Define your act-on and nutshell accounts information in config/app.php file.
# One can add both developer and client account information for both act-on and nutshell account.
# Both local and remote environment can be set inside config/app.php file.
